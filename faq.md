# Frequently Asked Questions

## General Questions

### What is Zaimea Groups?
Zaimea Groups is a comprehensive time tracking and project management platform designed to help teams and organizations streamline work hour reporting, project tracking, and team collaboration. It started as a personal tool and evolved into a full-featured progressive application.

### Is Zaimea Groups really free?
We offer a free plan with basic features that's perfect for individuals and small teams. For larger organizations or those needing advanced features like client portals, custom roles, and advanced reporting, we offer premium plans. You can start with our 30-day free trial to explore all features.

### What does "progressive application" mean?
We call Zaimea Groups "progressive" because it grows with you. We continuously add new features and improvements based on user feedback. If you need a specific feature that doesn't exist yet, you can request it and we'll work to implement it as quickly as possible.

### Do I need technical skills to use Zaimea Groups?
Not at all! Zaimea Groups is designed to be intuitive and user-friendly. You don't need to write code or have IT expertise. Our interface is straightforward, and we provide comprehensive documentation and support.

## Account & Billing

### How do I create an account?
You can create an account by visiting our [registration page](route('register')). After registration, you can either create a new group or join an existing one via email invitation. Make sure to review our [Terms of Service](route('terms.show')) and [Privacy Policy](route('policy.show')) before registering.

### Can I belong to multiple groups?
Yes! Each registered user can create multiple groups or belong to several groups simultaneously. This is perfect for freelancers, consultants, or anyone working with multiple teams or organizations.

### How do I change my account settings?
Account settings can be accessed through the user dropdown menu in the top-right corner of the application. From there, you can update your profile, change password, and manage notification preferences.

### What happens if I forget my password?
Use the "Forgot Password" link on the login page. We'll send you an email with instructions to reset your password. Make sure to check your spam folder if you don't see the email.

## Features & Usage

### What time tracking features does Zaimea offer?
Zaimea Groups offers comprehensive time tracking including:
- Start/end time recording
- Break tracking and notes
- Automatic monthly hour calculations
- Project and task association
- Flexible time formats (12/24 hour)
- PDF report generation

### Can I track time for specific projects and tasks?
Absolutely! We offer three tracking modes:

1. **Simple Mode** - Basic punch in/out without project association
2. **Project-based Mode** - Time associated with specific projects
3. **Project & Task Mode** - Detailed tracking with specific tasks (research, coding, testing, etc.)

### What kind of reports can I generate?
You can generate detailed monthly reports showing:
- Total hours worked
- Breakdown by project/task
- Client-specific reports
- Custom date range reports
- All reports can be exported as PDF for easy sharing

### Does Zaimea support night shifts?
Yes! We have a specific configuration option called "Allows nightshifts" that enables tracking for work periods that span across midnight (e.g., 10 PM to 2 AM).

### Can I plan future work or vacations?
Yes, with the "Future entries" configuration enabled, you can create time entries for future dates, which is perfect for planning vacations, scheduled work, or upcoming projects.

## Groups & Teams

### How do I create a new group?
After logging in, click on the user dropdown menu in the top-right corner and select "Create New Group". If you don't belong to any group, this option will automatically appear on your dashboard.

### What are the different user roles?
Zaimea Groups has 7 predefined roles:

| Role | Description |
|------|-------------|
| **Owner** | Full control over the group |
| **Top Manager** | Almost all permissions except owner-specific |
| **Manager** | Most management functions |
| **Co-manager** | Extended management capabilities |
| **Supervisor** | Limited management with approval rights |
| **Member** | Basic user with data entry rights |
| **Client** | External access to specific projects |

> {tip} You can also create custom roles with specific permissions tailored to your organization's needs.

### How do I invite team members?
Users with appropriate permissions (Co-manager, Manager, or Owner) can invite new members through the Group Panel. You'll need to enter their email address and assign them a role.

### Can I work with external clients?
Yes! With the [Clients plugin](/docs/groups/{{version}}/plugins#clients) enabled, you can:
- Create client accounts
- Assign them to specific projects
- Give them limited access to view relevant reports
- Control what data they can see

### What are plugins and how do they work?
Plugins are optional features that extend Zaimea's functionality:

| Plugin | Description |
|--------|-------------|
| **Charts** | Visual analytics for work hours |
| **Clients** | External client access management |
| **Locking** | Automatic record protection |
| **Monthly Quotas** | Work hour target tracking |
| **Report Approval** | Workflow management |
| **Templates** | Note templates with required fields |
| **Tasks** | Detailed task tracking |

## Technical Support

### What browsers are supported?
Zaimea Groups works on all modern browsers including:
- Chrome (latest version)
- Firefox (latest version)
- Safari (latest version)
- Edge (latest version)

> {note} We recommend using the latest browser version for the best experience and security.

### Is my data secure?
Security is our top priority. We implement:
- Industry-standard encryption (SSL/TLS)
- Regular automated backups
- Secure data centers with 24/7 monitoring
- GDPR compliance
- Comprehensive [Privacy Policy](route('policy.show'))

Currently, we're focusing on making our web application as robust as possible. API access is on our roadmap for future development. If this is critical for your needs, please [contact our team](route('contact')).
